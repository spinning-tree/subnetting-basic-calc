# Subnet Calc Pro

Calculadora personal de subredes.

## Características
* **Cálculo Circular**: Cambia CIDR, Hosts o Subredes y el resto se ajusta automáticamente.
* **Soporte de Clases**: Detecta automáticamente la clase de la IP (A, B, C) para calcular saltos de subred.
* **Visualización Binaria**: Desglose de máscara en 32 bits.

## Cómo usar
1. Introduce la **IP Base**.
2. Define tus necesidades: ¿Cuántos **Hosts** por subred o cuántas **Subredes** totales necesitas?
3. Obtén instantáneamente el **Network ID**, **Broadcast** y el rango utilizable.

## Tecnología
* **React 18** (UI Reactiva)
* **Tailwind CSS** (Estilos)
* **Babel Standalone** (Compilación en navegador)
